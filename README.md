## Hello GitHub!

- My Projects:
  - [Rubik's Cube with JOGL](https://github.com/TracksJosh/CS465-Project-Boddicker-Goff) ![progress](https://progress-bar.xyz/100/)
    - This is a group project developed by [AdaJoB](https://github.com/AdaJoB) and me for a Computer Graphics course.
  - [Quivia (Front End)](https://github.com/TracksJosh/FrontEnd) ![progress](https://progress-bar.xyz/100/)
    - This is the Front-End of my Senior Honors Project. It is a web page for an asynchronous multiplayer trivia game that uses questions stored in *OpenTDB* and *QBReader* databases.
  - [Quivia (Back End)](https://github.com/TracksJosh/FrontEnd) ![progress](https://progress-bar.xyz/100/)
    - This is the Back-End of my Senior Honors Project. It is a Tornado server for an asynchronous multiplayer trivia game that uses questions stored in *OpenTDB* and *QBReader* databases.
  - [Totally Accurate Minecraft Simulator (KTANE Module)](https://github.com/TracksJosh/ktaneTAMS) ![progress](https://progress-bar.xyz/100/)
    - This is my first modded *Keep Talking and Nobody Explodes* (KTANE) module, which is based on a *Minecraft* world I developed nearly four years before the release of this module.
  - [Literally Crying](https://github.com/TracksJosh/ktaneLiterallyCrying-master) ![progress](https://progress-bar.xyz/100/)
    - The original Literally Crying needy module is the origin of my profile picture on *GitHub*. 
  - [The Alteran Trail](https://github.com/TracksJosh/ktaneAlteranTrail-master) ![progress](https://progress-bar.xyz/100/)
    - This modded KTANE module is based on a computer game developed by my Algebra II Honors teacher, inspired by *The Oregon Trail* and *Stargate*.
  - [Shape Cipher](https://github.com/TracksJosh/ktaneShapeCipher-master) ![progress](https://progress-bar.xyz/100/)
    - This modded KTANE module features original substitution ciphers developed specifically for this project.
  - [Connected Quizbowl](https://github.com/TracksJosh/ktaneConnectedQuizbowl-master) ![progress](https://progress-bar.xyz/100/)
    - This modded KTANE module fetches Quiz Bowl questions from *QBReader*'s database. It uses a collection of questions in case the module has no internet access.
  - [Multi-Ruleseeded Modules](https://github.com/TracksJosh/ktaneMultiRuleseededModules-master)
    - This collection of modded KTANE modules requires the experts to use many different versions of manuals to solve them. The Fisher-Yates Shuffle algorithm determines the rules, maze, and combinations required to solve these sets of modules.
  - [Yahtzee](https://github.com/hennigesclaire/Yahtzee) ![progress](https://progress-bar.xyz/70/)
    - This is a group project developed by [hennigesclaire](https://github.com/hennigesclaire), [richardthegat](https://github.com/richardthegat), and me for a Software Engineering course. It is a Yahtzee game played in a Java Swing application.
  - [Totally Accurate Minecraft Simulator (Minecraft World)](https://www.planetminecraft.com/project/totally-accurate-minecraft-simulator/) ![progress](https://progress-bar.xyz/75/)
    - This *Minecraft* world features a game similar to *Totally Accurate Battle Simulator* by Landfall Games. Over 1,000 unit types from various time periods, pop culture, and fictional universes can be used to develop intricate armies. This world also follows a story through boss battles.
  - [The Cicada](https://www.youtube.com/watch?v=BhqZ2geHODM&t=32s) ![progress](https://progress-bar.xyz/50/)
    - This modded KTANE module is inspired by the cicada insect and Cicada3301, a group that developed cryptographic puzzles during the early-to-mid 2010's.
  - [Repository of Quizbowl](https://github.com/TracksJosh/repo_of_qb) ![progress](https://progress-bar.xyz/50/)
    - This is a website that should contain web pages for all answers to Quiz Bowl questions, which contain links to stock clues, obscure clues, *Wikipedia* articles, and *YouTube* videos explaining the answers.
